```
┌─────────────┐        ┌──────────────────┐        ┌──────────────┐
│  Ứng dụng   │───────▶│  Ứng dụng Proxy  │───────▶│  Máy chủ     │
│  (VD: YouTube)│◀───────│  (chặn & sửa)    │◀───────│  (trả response)│

                       Script sửa response
```

<summary><h3>⭐ Shadowrocket — Hướng dẫn đầy đủ (Khuyên dùng)</h3></summary>

---

#### 📦 PHẦN 1: Cài Module

**Bước 1.1** — Mở ứng dụng **Shadowrocket** trên iPhone/iPad

**Bước 1.2** — Nhấn vào tab **"Cấu hình"** (Configuration) ở thanh điều hướng phía dưới màn hình (biểu tượng trang giấy, tab thứ 2 từ bên phải)

**Bước 1.3** — Bạn sẽ thấy danh sách các file cấu hình. Nhấn vào file config đang sử dụng (file có dấu ✓ màu xanh bên cạnh)

**Bước 1.4** — Một menu hiện ra, chọn **"Sửa cấu hình"** (Edit Plain Text)

**Bước 1.5** — Kéo xuống dưới tìm dòng có chữ **`[Module]`**

**Bước 1.6** — Nhấn nút **＋** (dấu cộng) bên cạnh mục Module

**Bước 1.7** — Chọn **"Thêm URL Module"** (Add Module URL)

**Bước 1.8** — Dán đường link sau vào ô nhập:

```
https://raw.githubusercontent.com/ekaznyra/NguyenNgocAnhTu/main/Module/NguyenNgocAnhTu_Premium.module
```

**Bước 1.9** — Nhấn **"Lưu"** (Save)

**Bước 1.10** — Quay lại màn hình Module, đảm bảo toggle (công tắc) của module đã **BẬT** (hiện màu xanh) ✅

---

#### 🔐 PHẦN 2: Cài chứng chỉ MITM (BẮT BUỘC)

> ⚠️ **Nếu bỏ qua phần này, module sẽ KHÔNG hoạt động.** Chứng chỉ MITM cho phép ứng dụng proxy đọc và sửa dữ liệu mã hoá (HTTPS).

**Bước 2.1** — Mở Shadowrocket → nhấn tab **"Cài đặt"** (Settings) ở thanh dưới

**Bước 2.2** — Nhấn vào mục **"Chứng chỉ"** (Certificate)

**Bước 2.3** — Nhấn **"Tạo chứng chỉ mới"** (Generate New Certificate)
- Đợi vài giây để tạo xong

**Bước 2.4** — Nhấn **"Cài đặt chứng chỉ"** (Install Certificate)
- Ứng dụng Safari sẽ tự động mở ra

**Bước 2.5** — Safari hiện thông báo "Trang web này đang cố tải một cấu hình" → Nhấn **"Cho phép"** (Allow)

**Bước 2.6** — Thoát Safari, mở ứng dụng **Cài đặt** (Settings) của iOS

**Bước 2.7** — Bạn sẽ thấy dòng **"Đã tải về hồ sơ"** (Profile Downloaded) ngay dưới tên Apple ID → Nhấn vào đó
- Nếu không thấy: vào **Cài đặt** → **Cài đặt chung** → **VPN & Quản lý thiết bị**

**Bước 2.8** — Nhấn **"Cài đặt"** (Install) ở góc trên phải → Nhập mật khẩu iPhone → Nhấn **"Cài đặt"** lần nữa → **"Xong"**

**Bước 2.9** — Bây giờ cần **tin cậy** chứng chỉ. Vào:
- **Cài đặt** → **Cài đặt chung** → **Giới thiệu** (About)

**Bước 2.10** — Kéo xuống cuối cùng → Nhấn **"Cài đặt tin cậy chứng chỉ"** (Certificate Trust Settings)

**Bước 2.11** — Tìm chứng chỉ có tên Shadowrocket → **BẬT** toggle (chuyển sang màu xanh)

**Bước 2.12** — Hệ thống hỏi xác nhận → Nhấn **"Tiếp tục"** (Continue)

---

#### ✅ PHẦN 3: Kiểm tra xem đã hoạt động chưa

**Bước 3.1** — Quay lại Shadowrocket → Bật công tắc kết nối (toggle lớn ở trang chính)
- Biểu tượng VPN sẽ hiện trên thanh trạng thái

## 🔒 Locket Gold DNS — Giữ Gold khi tắt proxy

### 💡 Vấn đề là gì?

Bình thường khi tắt proxy, Locket sẽ kiểm tra lại đăng ký → phát hiện không có Gold → mất Gold. DNS Profile giải quyết vấn đề này bằng cách chặn luôn việc kiểm tra.

### 🔧 Cách hoạt động

```
┌─────────────────────────────────────────────────────────────────────┐
│                                                                     │
│  🟢 Khi proxy BẬT:                                                 │
│     Locket → Script sửa response → Gold = CÓ → Lưu vào bộ nhớ     │
│                                                                     │
│  🔴 Khi proxy TẮT:                                                 │
│     Locket → DNS chặn API kiểm tra → Dùng bộ nhớ cũ → Gold giữ ✅  │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

### 📋 Cách sử dụng

| Bước | Thao tác |
|:---:|:---|
| 1 | Tải file **`LocketGoldDNS.mobileconfig`** về iPhone |
| 2 | Cài đặt: iOS **Cài đặt** → **Cài đặt chung** → **VPN & Quản lý thiết bị** → Cài profile |
| 3 | **BẬT** proxy → Mở Locket → Đợi Gold hiện lên |
| 4 | **TẮT** proxy → Mở lại Locket → Gold vẫn còn ✅ |

## ⚙️ Tuỳ chỉnh YouTube

Bạn có thể thay đổi hành vi của YouTube bằng cách sửa phần `argument` trong module.

### Cấu hình mặc định

```json
{
  "lyricLang": "vi",
  "captionLang": "vi",
  "blockUpload": false,
  "blockImmersive": false,
  "debug": false
}
```

### Giải thích từng tham số

| Tham số | Ý nghĩa | Mặc định | Giá trị có thể dùng |
|:---|:---|:---:|:---|
| `lyricLang` | Ngôn ngữ dịch lời bài hát | `vi` | `vi` `en` `ja` `ko` `zh-Hans` `zh-Hant` `fr` `de` |
| `captionLang` | Ngôn ngữ dịch phụ đề video | `vi` | `vi` `en` `ja` `ko` `zh-Hans` `zh-Hant` `fr` `de` |
| `blockUpload` | Ẩn nút camera/tải lên | `false` | `true` (ẩn) / `false` (hiện) |
| `blockImmersive` | Ẩn phần immersive header | `false` | `true` (ẩn) / `false` (hiện) |
| `debug` | Bật ghi log để gỡ lỗi | `false` | `true` (bật) / `false` (tắt) |

### Cách sửa

<details>
<summary><b>Shadowrocket / LanceX</b></summary>

1. Vào **Cấu hình** → Nhấn config đang dùng → **Sửa cấu hình**
2. Tìm dòng chứa `argument=` trong phần YouTube
3. Sửa nội dung JSON theo ý muốn
4. Nhấn **Lưu**

</details>

<details>
<summary><b>Surge</b></summary>

1. Vào **Modules** → Nhấn vào module → **Sửa** (Edit)
2. Tìm dòng `argument=` → Sửa JSON
3. Nhấn **Lưu** (Save)

</details>

<details>
<summary><b>Loon / Quantumult X / Stash / Egern</b></summary>

1. Vào phần quản lý Plugin/Module/Override
2. Nhấn vào module đã cài → Tìm phần argument
3. Sửa JSON theo ý muốn → Lưu

</details>
